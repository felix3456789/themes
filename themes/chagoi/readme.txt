== Description ==

Chagoi WordPress Theme is a clean blogger theme with full of features. Chagoi has different layouts and post formats and also compatible with WooCommerce and Elementor page builder plugins. If You want to create a clean site with unique design, Chagoi will gives the spirit to it. Check out the theme page for more functions!

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Chagoi in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
